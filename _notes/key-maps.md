### Visual Studio for Mac 주요 단축키

- **솔루션 탐색기로 포커스 이동**: `⇧⌘E` (Shift + Command + E)
- **편집기 창으로 다시 포커스 이동**: `⇧⌘E` (Shift + Command + E)
- **터미널 창 열기/닫기**: `⌃~` (Control + Tilde)
- **새로운 터미널 열기**: `⌃⇧~` (Control + Shift + Tilde)
- **다음 터미널로 전환**: `⌘]` (Command + ])
- **이전 터미널로 전환**: `⌘[` (Command + [)
- **다음 도구 창(패드)으로 전환**: `⌃⇥` (Control + Tab)
- **이전 도구 창(패드)으로 전환**: `⌃⇧⇥` (Control + Shift + Tab)
- **오른쪽 탭으로 이동**: `⌘⌥→` (Command + Option + Right Arrow)
- **왼쪽 탭으로 이동**: `⌘⌥←` (Command + Option + Left Arrow)
- **최근 파일 및 창 목록 보기**: `⌃⇥` (Control + Tab)

### 편집기 창 극대화

IntelliJ의 `Shift+Cmd+F12`와 같이 모든 도구 창을 한 번에 숨기는 단축키는 Visual Studio for Mac에 기본으로 존재하지 않습니다. 대신 아래와 같이 각 창을 개별적으로 숨겨 비슷한 효과를 낼 수 있습니다.

- **솔루션 탐색기(사이드바) 숨기기/보이기**: `⇧⌘E` (Shift + Command + E)
- **터미널(하단 패널) 숨기기/보이기**: `⌃~` (Control + Tilde)

### 터미널

- **터미널 탭 나누기**: `⌘\` (Command + \)
- **분할된 탭 간 이동**: `⌥⌘` + `방향키` (Option + Command + Arrow Keys)
- **터미널 간 이동**: `⇧⌘` + `[` 또는 `]` (Shift + Command + [ or ])
